# GY-87 IMU Module

Reads data from GY-87 IMU module containing MPU6050 (accelerometer/gyroscope), QMC5883L (magnetometer), and BMP180 (barometric pressure). Outputs all sensor data to Serial Monitor.

Code: [`../projects/26-gy87.ino`](../projects/26-gy87.ino)

## Required Components

| Component | Quantity | Notes |
|-----------|----------|-------|
| Arduino Uno R4 WiFi | 1 | - |
| GY-87 IMU Module | 1 | Multi-sensor module |
| Breadboard | 1 | - |
| Jumper wires | 4 | - |

**Libraries Required:** 
- Adafruit MPU6050
- QMC5883LCompass
- Adafruit BMP085 Library

## Wiring

- GY-87 VCC → 5V
- GY-87 GND → GND
- GY-87 SDA → SDA (I2C)
- GY-87 SCL → SCL (I2C)

| Component | Arduino Pin |
|-----------|-------------|
| GY-87 I2C | SDA/SCL |
| GY-87 VCC | 5V |
| GY-87 GND | GND |
