# RFID-RC522 Module

Reads RFID cards and tags using MFRC522 module. Displays card type and UID (Unique Identifier) to Serial Monitor when a card is detected.

Code: [`../projects/23-mfrc522.ino`](../projects/23-mfrc522.ino)

## Required Components

| Component | Quantity | Notes |
|-----------|----------|-------|
| Arduino Uno R4 WiFi | 1 | - |
| MFRC522 RFID Module | 1 | RFID reader |
| Breadboard | 1 | - |
| Jumper wires | 7 | - |

**Library Required:** MFRC522

## Wiring

- MFRC522 SDA → Pin 10
- MFRC522 SCK → Pin 13
- MFRC522 MOSI → Pin 11
- MFRC522 MISO → Pin 12
- MFRC522 RST → Pin 9
- MFRC522 3.3V → 3.3V
- MFRC522 GND → GND

| Component | Arduino Pin |
|-----------|-------------|
| MFRC522 SDA (SS) | 10 |
| MFRC522 SCK | 13 |
| MFRC522 MOSI | 11 |
| MFRC522 MISO | 12 |
| MFRC522 RST | 9 |
