# Soil Moisture Sensor

Measures soil moisture levels using an analog soil moisture sensor. Outputs raw analog values (0-1023) to Serial Monitor.

Code: [`../projects/03-moisture.ino`](../projects/03-moisture.ino)

## Required Components

| Component | Quantity | Notes |
|-----------|----------|-------|
| Arduino Uno R4 WiFi | 1 | - |
| Soil Moisture Sensor | 1 | Analog moisture sensor |
| Breadboard | 1 | - |
| Jumper wires | 3 | - |

## Wiring

- Sensor VCC → 5V
- Sensor GND → GND
- Sensor A0 → A0

| Component | Arduino Pin |
|-----------|-------------|
| Sensor A0 | A0 |
| Sensor VCC | 5V |
| Sensor GND | GND |
