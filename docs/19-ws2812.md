# WS2812 RGB LEDs Strip

Controls a chain of addressable RGB LEDs (NeoPixels) using FastLED library. Lights up each LED sequentially in blue color.

Code: [`../projects/19-ws2812.ino`](../projects/19-ws2812.ino)

## Required Components

| Component | Quantity | Notes |
|-----------|----------|-------|
| Arduino Uno R4 WiFi | 1 | - |
| WS2812B LED Strip | 1 | Addressable RGB LEDs |
| Resistor 470Ω | 1 | Data line protection (optional) |
| Capacitor 1000µF | 1 | Power supply filtering (optional) |
| Breadboard | 1 | - |
| Jumper wires | 3 | - |
| External 5V power supply | 1 | For longer strips |

**Library Required:** FastLED by Daniel Garcia

## Wiring

- WS2812 VCC → 5V (or external power)
- WS2812 GND → GND
- WS2812 DIN → Pin 6
- 470Ω resistor: DIN → Pin 6 (optional, for protection)

| Component | Arduino Pin |
|-----------|-------------|
| WS2812 DIN | 6 |
| WS2812 VCC | 5V |
| WS2812 GND | GND |
