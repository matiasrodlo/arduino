# MPR121 Capacitive Touch Sensor

Detects touch on up to 12 capacitive pads using MPR121 sensor. Outputs touch and release events to Serial Monitor for each pad.

Code: [`../projects/24-mpr121.ino`](../projects/24-mpr121.ino)

## Required Components

| Component | Quantity | Notes |
|-----------|----------|-------|
| Arduino Uno R4 WiFi | 1 | - |
| MPR121 Module | 1 | 12-channel capacitive touch |
| Breadboard | 1 | - |
| Jumper wires | 4 | - |

**Library Required:** Adafruit MPR121

## Wiring

- MPR121 VCC → 3.3V
- MPR121 GND → GND
- MPR121 SDA → SDA (I2C)
- MPR121 SCL → SCL (I2C)

| Component | Arduino Pin |
|-----------|-------------|
| MPR121 I2C | SDA/SCL |
| MPR121 VCC | 3.3V |
| MPR121 GND | GND |

**I2C Address:** 0x5A (default, configurable)
